# Beitragen zu JF-Coach

Danke für dein Interesse, zu JF-Coach beizutragen!

## Wie kann ich helfen?

### Bugs melden
Bitte nutze die [GitHub Issues](../../issues) und wähle das Template "Bug Report".  
Beschreibe:
- Was hast du erwartet?
- Was ist stattdessen passiert?
- Welches Gerät / Browser verwendest du?

### Features vorschlagen
Öffne ein [Issue](../../issues) mit dem Template "Feature Request" und beschreibe die gewünschte Funktion und warum sie hilfreich wäre.

### Code beitragen
1. Fork das Repository
2. Erstelle einen Branch: `git checkout -b feature/mein-feature`
3. Committe deine Änderungen: `git commit -m 'Add: kurze Beschreibung'`
4. Push zum Branch: `git push origin feature/mein-feature`
5. Öffne einen Pull Request

## Commit-Nachrichten

Bitte verwende einen klaren Präfix:
- `Add:` für neue Features
- `Fix:` für Bugfixes
- `Docs:` für Dokumentationsänderungen
- `Refactor:` für Code-Umstrukturierungen

## Verhaltenskodex

Bitte behandle alle anderen Mitwirkenden respektvoll. Dieses Projekt ist ein Gemeinschaftsprojekt der Jugendfeuerwehr-Community.
